# Blog / article by Jenny

Blogg site where you can create new bloggs as a login member and edit or destroy them. Also its possible to add comments to the posts.

### Ruby version ruby 2.5.1p57

### System dependencies

- Bundler 2.1.2
- Rails 6.0.2
- postgres DB
- Devise
- FactoryBot

### test suite

Semaphore
